## ğŸ§  Fitur Utama
1. **Login multi-role**  
   Role: `admin`, `teknis`, `administrasi`, `keuangan`.  
   Autentikasi sederhana pakai session (boleh plaintext password dulu).

2. **CRUD dasar proyek**  
   - Tampilkan daftar proyek dari tabel `projects`.
   - Setiap proyek memiliki kode, nama, lokasi, tanggal, dan deskripsi.

3. **Penilaian (Weighted Product)**  
   - Tiap bidang memiliki kriteria & bobot sendiri.  
   - User dengan role tertentu menilai semua proyek dengan skor numerik.  
   - Hasil dikalkulasi otomatis dengan rumus Weighted Product (WP).

4. **Agregasi Hasil (Metode BORDA)**  
   - Ranking WP dari ketiga bidang digabung pakai metode BORDA tertimbang.  
   - Tampilkan proyek prioritas berdasarkan total skor BORDA tertinggi.

5. **UI & Desain Modern**  
   - Gunakan Bootstrap 5 + CSS gradient modern (teal, indigo, amber).  
   - Gunakan animasi CSS ringan (hover, fade, pulse).  
   - Responsif dan terlihat profesional seperti dashboard web modern.

6. **Database yang Dibutuhkan**
   - `users(id, username, password, fullname, role)`
   - `projects(id, code, name, location, date_offer, description)`
   - `criteria(id, part, name, weight, type)`
   - `evaluations(id, project_id, user_id, criteria_id, score)`
   - `part_weights(part, weight)`
